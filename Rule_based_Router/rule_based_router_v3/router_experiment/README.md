# Rule-based Router V3

## Phase 0

Phase 0 là baseline độc lập được sao chép cơ học từ Rule-based Router V2.
Mục tiêu là xác nhận pipeline service, adapter và evaluation có thể tích hợp
V3 mà không thay đổi hành vi của V2. Các cải tiến thuật toán sẽ được đưa vào
các phase tiếp theo và phải có benchmark/regression riêng.
